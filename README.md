# Awesome Project

This is how your project should look. It should be public, include a license, README, and a .gitignore.

## Delivering

Delivering project milestones should be made using [Releases](https://docs.github.com/en/repositories/releasing-projects-on-github/managing-releases-in-a-repository)
and the deliverables should not be in your repository, but instead contained within the release.
